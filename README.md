# AI-Powered Drug Discovery System

## Overview
A deep learning–based drug–target interaction prediction system designed to assist in early-stage drug discovery. The project includes an interactive Streamlit dashboard for real-time prediction visualization.

## Tech Stack
- Python
- Deep Learning
- Streamlit
- Pandas, NumPy
- Scikit-learn / TensorFlow (if used)

## Key Features
- Drug–target interaction prediction
- Model training and inference
- Interactive Streamlit dashboard
- Modular and scalable code structure

## How to Run
pip install -r requirements.txt  
streamlit run app.py

## Use Case
Supports researchers and students in understanding AI-powered drug discovery workflows.
